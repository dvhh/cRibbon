cRibbon : Protein Ribbon Viewer

Overview

cRibbon is a simplistic protein viewer in Cartoon representation implemented in javascript, it aim to have a wide browser compatibility, As long as your browser can use the <canvas> element [1] [2], And can be accelerated with webGL when available [1] [2].
To achieve this feature it relies on the popular THREE.js 3D abstraction library.

Motivations

The initial goal was to provide a basic javascript PDB viewer, other software 3D engine were considered, but THREE.js was selected but its drop in evolution to webGL when implemented, as well as other display methods.
Various Java exploit and the growth of mobile equipement accelerated the need for a full javascript implementation.
Of course, such implementations already exist, but have generally Desktop class processor requirement.

Limitations

  • The ribbons drawn are not so smooth : the ribbons use a flat shading model to accelerate the rendering while keeping a sufficient fidelity
  • Rendering large PDB files is slow, while not implemented yet a polygon reduction feature is planned to further accelerate the rendering
  • the viewing style is currently limited to this light "cartoon view" an atom view is considered

Roadmap as of 2013-04

  • Usage Simplification : The current implementation requires a lot of code to get started, future development should require the user to use 1 line of code, while keeping the other functions for alowing more complex case scenario
  • Combine all the scripts in one, to reduce the number of http call and provide better gzip compression when available
  • Polygon reduction algorithm : Javascript polygon reduction can be quite slow depending on the number of vertices
  • Script for server side logic and generation : we could lighten the initial work of translating the PDB files to model on the server side
